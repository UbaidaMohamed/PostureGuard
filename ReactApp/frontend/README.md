# PostureCorrection frontend

## Project info

This is the frontend for the PostureCorrection (PostureGuard) application.

## How can I edit this code?

You can work locally using your preferred IDE. The only requirement is Node.js and npm/yarn installed.

Quick start:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>/frontend

# Step 3: Install the necessary dependencies.
npm install

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How can I deploy this project?

Build the app and serve the `dist/` folder with any static host. Example:

```sh
npm run build
npm run preview
```

If you need help deploying to a specific platform (Netlify, Vercel, etc.) let me know.
